# 📨 예약 확정 알림 (Gmail 이메일 & 카카오톡 알림톡) 연동 가이드

예약 확정(`status: 'Confirmed'`) 시, 예약자의 이메일(Nodemailer + Gmail SMTP) 및 카카오톡 메시지(Solapi)로 자동 알림을 발송하기 위한 연동 구조가 완료되었습니다.
아래 절차에 따라 계정 설정 및 환경변수 입력을 마쳐주시면 즉시 정상 작동합니다.

---

## 1단계. Gmail (이메일) SMTP 연동 절차

도메인 구매 없이 본인의 개인 Gmail 계정을 이용하여 고객에게 예약 확정 메일을 발송합니다.

### 1. 구글 앱 비밀번호 발급
1. 메일을 발송할 **구글 계정**에 로그인한 뒤, [구글 계정 관리](https://myaccount.google.com/)로 이동합니다.
2. 왼쪽 메뉴에서 **보안(Security)** 탭을 선택합니다.
3. 'Google에 로그인하는 방법' 섹션에서 **2단계 인증(2-Step Verification)**을 설정(필수)합니다.
4. 2단계 인증 페이지 맨 아래에 있는 **앱 비밀번호(App passwords)** 메뉴를 선택합니다.
5. 앱 이름에 `더 헤어 갤러리 예약 시스템` 등 알아보기 쉬운 이름을 기입하고 **만들기**를 누릅니다.
6. 화면에 생성되는 **16자리 비밀번호 코드**를 복사해 둡니다. (다시 볼 수 없으므로 바로 메모장에 저장하세요.)

### 2. 환경 변수 설정
`/.env.local` 파일에 발송할 본인의 Gmail 주소와 위에서 발급받은 16자리 앱 비밀번호를 설정합니다.
```env
# Gmail 발신 주소
GMAIL_USER=your_gmail_address@gmail.com

# 구글에서 발급받은 16자리 앱 비밀번호 (공백 없이 입력)
GMAIL_APP_PASSWORD=abcdefghijklmnop
```

## 2단계. Solapi (카카오톡 알림톡) 연동 절차

도메인 없이 휴대폰 번호 기반으로 고객의 카카오톡으로 예약 확정 메시지를 보냅니다.

### 1. Solapi 및 카카오톡 채널 준비
1. **Solapi 가입**: [Solapi 공식 홈페이지](https://solapi.com)에서 가입을 진행합니다.
2. **카카오톡 채널 개설**: dlfeks 에서 채널을 만들고 비즈니스 채널로 인증받습니다.
3. **채널 연동**: Solapi 콘솔의 **카카오톡 채널 관리** 메뉴에 본인의 채널을 등록하여 `플러스친구 ID(pfId)`를 발급받습니다.
4. **발신번호 등록**: Solapi 콘솔에서 발신번호(예: 미용실 전화번호 혹은 개인 휴대폰 번호) 인증을 완료합니다.

### 2. 알림톡 템플릿 등록 및 승인
카카오 알림톡은 사전 승인된 템플릿만 보낼 수 있습니다. Solapi의 알림톡 템플릿 관리 메뉴에서 아래 예시를 참고하여 템플릿을 등록해 주세요.

* **템플릿 이름 예시**: `예약 확정 안내`
* **템플릿 내용 예시**:
  > **[더 헤어 갤러리] 예약 확정 안내**
  >
  > #{고객명}님, 요청하신 예약이 정상적으로 확정되었습니다.
  >
  > • 예약일시: #{예약일시}
  > • 시술명: #{시술명}
  > • 결제금액: #{결제금액}
  >
  > 예약 시간 10분 전까지 매장에 방문해 주시기 바랍니다. 감사합니다.
* **⚠️ 주의**: 템플릿 승인 신청 시 본문 속 변수(`#{고객명}`, `#{예약일시}`, `#{시술명}`, `#{결제금액}`)가 `lib/kakao.ts` 코드 내부의 `variables` 키명과 대소문자/띄어쓰기까지 완벽하게 일치해야 변수 치환이 정상적으로 작동합니다.

### 3. 환경 변수 설정
템플릿 승인이 나면 Solapi 콘솔에서 API Key와 발신 상세 정보를 복사하여 `/.env.local` 파일에 추가합니다.
```env
# Solapi API Credentials (콘솔에서 발급)
SOLAPI_API_KEY=YOUR_SOLAPI_API_KEY
SOLAPI_API_SECRET=YOUR_SOLAPI_API_SECRET

# Solapi에 등록 및 인증 완료한 발신자 번호 (하이픈 제외 숫자만)
SOLAPI_FROM_PHONE=01012345678

# 카카오 채널 ID (KA01PF... 형태)
SOLAPI_PF_ID=KA01PFxxxxxxxxxxxx

# 승인 완료된 알림톡 템플릿 ID (KA01TP... 형태)
SOLAPI_TEMPLATE_ID=KA01TPxxxxxxxxxxxx
```
